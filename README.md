# Meu-Portfolio

Este repositorio contém meu portifólio online e este README a descrição de seus pertences:

## Estrutura HTML

`index.html`: É a página inicial que levará para as outras páginas \
`Sobre.html`: É a pagina que contém as principais informações sobre mim mesmo bem como meus contatos \
`Projetos.html`: É a pagina que contém meus projetos atuais no qual trabalho

## Estrutura CSS

`style1.css`: É o arquivo css responsável por editar o `index.html` \
`style2.css`: É o arquivo css responsável por editar o `Sobre.html` \
`style3.css`: É o arquivo css responsável por editar o `Projetos.html` 

## Funcionalidade JS

O portfólio possui um icone no canto superior direito que quando clicado mudará o tema de cores da página
